   - On Windows: You can use Command Prompt or Git Bash.

   Use the `cd` command to change to the directory where you want the project to be located. For example:
   ```bash
   cd path/to/your/folder
   ```

   Run the following command to clone the repository:
   ```bash
   git clone https://github.com/ankitchaurasiya84/amazon-clone.git
   ```

4. **Navigate to the Cloned Repository**

   ```bash
   cd amazon-clone
